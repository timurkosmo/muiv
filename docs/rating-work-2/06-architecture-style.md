# Архитектурный стиль

## Выбранная архитектура
**Гибридная архитектура** (микросервисная + слоистая + событийно-ориентированная)

## Компоненты архитектуры

### Микросервисы
1. Сервис управления документами
2. Сервис маршрутизации
3. Сервис поиска и индексации (Elasticsearch)
4. Сервис отчетности
5. Сервис уведомлений
6. Сервис аутентификации и авторизации
7. Сервис хранения файлов

### Инфраструктурные компоненты
- API Gateway
- RabbitMQ (шина сообщений)
- Service Discovery
- Circuit Breaker
- Distributed Tracing
- Централизованное логирование

### Контейнеризация и оркестрация
- Docker
- Kubernetes
